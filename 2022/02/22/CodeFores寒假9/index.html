2022.02.20 四题 day9

Codeforces Round #772 (Div. 2) A-D

好久没这样爽加91分了

舒服了

A

思路

每次可以换两个数，前后的或结果相等。

或结果相等意味着每一位上如果出现1就要保留

最小意味着只保留1个就行

Code

B

思路

进行几次更改操作，使得结果不存在区间最大值

每次遇到区间最大值，把ai+1更换为max(ai+2,ai)

Code

C

思路

把ax换成ay-az 不超过n次操作

首先如果已经不递减 YES

因为x<y<z 把yz选为最右边两个，然后把其他全部换成ay-az

  1. 最右边俩保证递增
  2. 最右边一个不小于0

Code

D

思路

2x+1 4x 分别可以看成 位数+1 和+2

记f[i] 从0->i位有多少个数

f[i]=f[i-1]+f[i-2]

f[0]=f[1]=1 都只有其本身

再把f前缀和处理。

先把a数组排序，如果一个数是可以由之前的数产生，就不计算他了。

不断进行还原的操作，利用Set保存集合，相当于去重

如果一个数是独立的，计算他在2^p范围内等产生的派生数。
