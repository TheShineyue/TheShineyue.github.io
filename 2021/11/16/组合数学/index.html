第一类斯特林数

S1(n,m) n->m个圆
S1(n,n)=1,S1(n,0)=0
S1(n,m)=(n-1)S1(n-1,m)+S1(n-1,m-1) (n>1,m>1)
S1(n,1)=(n-1)!
S1(n,n-1)=n(n-1)/2
S1(n,m)>=S2{n,m}

第二类斯特林数

n->m个空
S2{n,m}=S2{n-1,m-1}+S2{n-1,m}*m,n>=1
n<m S2{n,m}=0
n>=1 S2{n,1}=1
n>=2 S2{n,2}=2^(n-1)-1

隔板加抽屉

HDU - 1205

题面

HOHO，终于从Speakless手上赢走了所有的糖果，是Gardon吃糖果时有个特殊的癖好，就是不喜欢将一样的糖果放在一起吃，喜欢先吃一种，下一次吃另一种，这样；可是Gardon不知道是否存在一种吃糖果的顺序使得他能把所有糖果都吃完？请你写个程序帮忙计算一下。

思路

计算得最多的一种糖果 x
剩下的糖果数总和为 y

计算得一种符合条件的排列方法：
把x个糖果排列好，把剩下的糖果作为隔板插入产生的x-1个空里面
此时由于抽屉原理，需要x-1个糖果即可
多余x-1的糖果可以再次看成把一个小规模的糖果插入一个大规模的已排列好的糖果排列中
即y>=x-1输出Yes

下面证明必要性：
当y<x-1时 x必有两天连着吃

第二类斯特林数||容斥

HDU - 6143

题意

要求从m个字符中拿一定字符 分给两段n n
前n和后n的字符不能有重复
求方案数

思路

遍历i从1到m
C(m,i)*dp (i,n) *(m-i)^n

其中的DPi n代表把i个字符可重复的放入n位中的方法数

DP i n = (DP i-1 n +DP i-1,n-1)* i
//why 不是 DP i n =(DP i-1 n +DP i n-1 +DP i-1 n-1)
//好像会有重复，之后好好想想
想到了，第二类斯特林数

第二类斯特林数+插板法

HDU - 4045

题意

n r k m

把n个机器取r个，两两编号之差大于等于k，r个分成i组（i在1到m）

思路

其实就是俩问题，① 把n个正确取出r个，② 把r分成i组，第二类斯特林数
主要是前面的那个问题，两两编号之差大于等于k,所以每k取一个，这样就有(r-1)k+1个必须的机器了
此时如果发现n<(r-1)k+1则无解
然后多余的机器为 n-((r-1)k+1)
多余和原来要取出的r个一起插入r个位置
C((r-1)k+1+r,r)

②也不必多说，裸的板子
