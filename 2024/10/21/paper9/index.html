论文阅读 CliMedBench: A Large-Scale Chinese Benchmark for Evaluating Medical Large Language Models in Clinical Scenarios

Abstract

  • CliMedBench，一个全面的基准与14个专家指导的核心临床场景，专门设计来评估LLMs跨越7个支点的医学能力。它包含33,735个派生问题来自顶级医院的真实医疗报告和真实的检查报告。该基准的可靠性在几个方面得到了证实。
    发现的问题：
  • 中医：中文医学法学硕士在这一基准上表现不佳，特别是在医学推理和事实一致性至关重要的情况下，这凸显了临床知识和诊断准确性进步的必要性。
  • 几个通用领域：一些通用领域的 LLM 在医学临床中表现出巨大的潜力，而许多医学 LLM 的输入能力有限阻碍了它们的实际应用。这些发现揭示了 LLM 在临床场景中的优势和局限性，并为医学研究提供了重要的见解。

Introduction

  • 原有的中文医学LLM旨在通过提供诊断帮助和治疗建议来解决复杂的医疗问题。尽管如此，由于缺乏对其性能的全面和系统评估（包括响应准确性、幻觉发生率和内容安全等方面），阻碍了它们融入临床实践。

  • 目前的中文基准来自于开放教育资源，这可能不够实践性和可能存在数据污染：基于真实医学案例的基准具有更高的真实性和异质性，同时更准确地反映了临床实践中遇到的复杂性。这些案例带来了更大的挑战和复杂性，导致对实际应用中模型性能和稳健性的评估更加严格，包括临床决策支持、诊断和治疗建议。此外，从开放资源开发的基准容易受到数据污染问题的影响。

    1. 基于整合了教科书、学术文章和资格考试等可访问来源的数据的MedQA和MedMCQA，此类评估基准的有效性存在争议：医学考试是低效的临床表现指标

    2. 基于 EHR 的大规模基准，如 emr-QA 已经解决了临床 QA 的不足；然而，语言差异阻碍了它们直接适用于评估中文医学法学硕士。

    3. 中文基准，包括CMExam、CMB和 MLEC-QA，主要从 CNMLE 和 NMLEC 等考试中获取数据。 MedBench使用考试题目和人工生成的 EHR 来评估不同子学科的 LLM 考试解决能力。
      尽管这些基准进行了全面分析，但由于缺乏真实世界的医疗案例数据，因此与实际医疗实践脱节，并且使用基于考试的数据引发了对数据污染的担忧。

  • 为了解决先前研究的局限性，我们引入了 CliMedBench，这是一个强大的基准，包含 14 个核心医学场景中的 33,735 个问题，用于评估法学硕士在六个维度上的能力，主要来自真实案例，以符合医学标准和实践。

    1. 中国医学法学硕士在此基准上表现不佳，尤其是在医学推理和事实一致性至关重要的情况下，这凸显了临床知识和诊断准确性需要提高。
    2. 几个通用领域的法学硕士在医学临床中表现出巨大潜力，而许多医学法学硕士的输入能力有限阻碍了它们的实际应用。
    3. 医学背景中固有的不确定性会严重损害模型生成响应的准确性。

The Proposed Benchmark

The Taxonomy of CliMedBench

结构良好的分类法使我们能够对医学法学硕士进行更细致的评估，同时确保评估全面且具有实践意义。
我们的分类法旨在保持基准在现实世界临床场景中的适用性和全面性，它基于反映医疗实践并完全涵盖医疗实践的分类

“Who-What-How”方案对现实世界的临床医疗实践进行分类，提供 14 个核心临床场景供评估。

  1. 沿着“Who”轴，我们区分了医学领域的五个主要角色：放射技师、药剂师、患者、医学生和专科医生，其中医生包括主治医师、外科医生和其他医学专家。
  2. “What” 涵盖了广泛的关键医疗场景，涵盖了基础知识测试、住院诊断、临床路径推理、病例总结、错误治疗检测等。这使得 CliMedBench 能够从七个角度评估法学硕士的医疗能力，包括临床问答、知识应用、推理、信息检索、总结能力、幻觉和毒性。

住院诊断跨越四个时期，涵盖了从入院到出院的患者护理全过程

  • ID #1 是指医疗专业人员和放射技师对检查的选择。
  • ID #2 涉及医生的诊断，将检查结果与患者的病史和其他健康数据相结合。
  • ID #3 期间，与药剂师和医务人员合作制定治疗策略，从药物干预到外科手术。
  • ID #4 涉及医生向患者提供出院指导

Construction and Statistics

CliMedBench 源自中国顶级三级医院的真实电子健康记录 (EHR)，并辅以考试练习、医疗指南、教科书、学术文章和人工注释的在线咨询。

该语料库涵盖众多医学专业，经过精心策划，以增强 CliMedBench 的多样性。

  • EHR 数据由我们的合作医院提供。该数据集包括 2023 年 1 月 1 日至 2023 年 3 月 31 日期间包含放射记录的所有 EHR。
  • 考试问题来自 NM LEC 2023 年度考试，涵盖外科和内科领域的 16 个科目。

说明：

  1. 在构建数据集之前，所有 EHR 都已由伦理委员会和专家双重去身份化，以确保不会泄露患者或医疗保健专业人员的 PHI

（隐私性）

  1. 我们对原始数据进行预处理，以过滤低质量数据，并通过正则表达式匹配和医疗实体识别等自动技术确保格式正确

（格式正确保证）

构建数据集的工作流程：

  1. 医疗专家最初根据 EHR 中医生笔记所指示的内容类型制定指南

  2. 这些指南用于将 EHR 分割成句子，然后由 LLM1 将其分类为问题生成的组件，通过将正确答案与来自不同但主题相似的 EHR 片段的干扰项相结合来制定 QA 对

  3. 二级 LLM（LLM2）审核数据集是否存在不一致或歧义，并由医疗专家评估标记的项目，以确保仅解决相关问题。来自此评估的反馈改进了 LLM1 的处理以减轻反复出现的数据集缺陷。

  4. 数据集经过迭代验证周期，确保至少 90% 的问题符合质量标准，并由双重专家评审确认。我们最终获得了 14 个核心临床场景的 33,735 个实例，这些实例严格基于医生笔记和临床治疗记录。

CliMedBench 的数据分布，涵盖了 19 个医学分支，例如神经外科和胃肠病学。

类型包括：

• 多项选择临床问答。
• 排序问题，例如手术步骤重新排序。
• 开放式生成，例如出院总结、主观临床问答。

Characteristics

CliMedBench 在多个方面优于现有基准：

  1. 真实性和独特性，因为它通过专门使用来自顶级医院的专家注释的 EHR，真实地反映了医生的实践经验，具有最新、真实的视角，同时降低了数据污染的可能性。
  2. 全面性和多维性，因为它经过精心设计，以符合中国临床实践，涵盖了具有多模式信息的多种医学学科，提供了广泛的评估视角。
  3. 实用性，因为它提供了一种新颖的基于代理的计算机化自适应测试方法，以确保使用 CliMedBench 进行快速评估。

Models and Evaluation Metrics

为了评估最先进的水平，我们使用 CliMedBench
对来自普通领域和医学领域的 11 名代表性LLM进行评估

  • 多项选择题： acc
  • 排序题：Kendall’s τ
  • 开放性生成：专家级人工评估与补充自动评估指标相结合

例如，ROUGE-1或者基于 SimCSE 的相似度 ， 使用不同的医学文档对 SimCSE进行微调，然后利用生成的模型导出句子向量，最后计算与参考文献的语义相似度。

Main Results

我们还聘请了人类专家来评估四个维度的开放式生成（WTD 和多模态报告分析），
包括医学正确性、完整性、流畅性和友好性。

中国医学法学硕士在这一基准上表现不佳，尤其是在医学推理和事实一致性至关重要的情况下。

  • API 构建的模型通常优于其他模型，平均分数超过 50
  • ERNIE-Bot、GPT-4 和 Qwen 的平均分数相当接近，分别为 69.2、69.0 和 68.5
  • 目前的医学法学硕士表现明显不足：即使是表现最好的 MedicalGPT 也只获得了 38.7 的平均分数。这一缺陷主要源于这些法学硕士的语言理解能力不达标。
  • 几门中国法学硕士（ERNIE-Bot 和 Qwen）在中国临床医学方面的表现与 GPT-4 相当，得分主要在 68.5 到 69.2 之间。这可能源于独特的治疗方法、表达方式和中国制造的药品，与 GPT 系列模型的训练数据不同。这些中国法学硕士与 GPT 系列之间的能力差异主要体现在医学知识和推理方面。

我们将总结 LLM 在特定评估维度上的表现：

  • 在临床 QA 能力方面，Qwen 的表现优于其他模型，平均得分为 88.7
  • 模型性能在不同场景下的变化是显而易见的，ChatGPT 在 ID #2 上获得了最高分（97.4），但在其他场景中并未名列前茅
  • GPT4 和 ERNIE-Bot 表现出卓越的推理能力，平均得分分别为 75.1 和 73.3。通用和医学特定 LLM 之间显著的表现差异凸显了进一步增强医学特定 LLM 推理能力的必要性。
  • FIT 数据旨在通过合并错误的参考来触发幻觉。他们的数据源与BKT相同，但模型准确率明显下降，从平均47.3分跌至8.3分。这一大幅下降表明LLM学生无法不加批判地采纳输入中的观点，亟待改进。
  • WTD数据集上表现出的幻觉表明，对于特殊结构的问题，LLM学生不仅需要掌握问题考察的知识点，还需要理解问题中的逻辑关系，这可能超出模型的能力。
  • 领先的一般LLM学生的知识应用能力从79.8分到82.4分不等，表明这些模型拥有丰富的医学知识。
  • 在信息检索任务上，MedicalGPT显著缩小了与领先LLM学生的差距，得分为33.3分，仅比表现最好的ERNIE-Bot低8.7分。这种改进主要源于其生成的术语的专业性。
