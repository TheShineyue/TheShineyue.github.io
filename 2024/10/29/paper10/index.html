论文阅读 Image Textualization An Automatic Framework for Creating Accurate and Detailed Image Descriptions

Abstract

  • 图像描述数据集的来源：

    • 从网络上抓取的图像-文本对：尽管这些描述丰富多彩，但通常质量较差且噪声较大
    • 人工标记：诸如 COCO 之类的数据集通常非常简短且缺乏细节。

  • 虽然详细的图像描述可以由人工注释，但高昂的注释成本限制了可行性。这些限制强调了需要更高效、可扩展的方法来生成准确、详细的图像描述

  • 提出了Image Textualization IT 的创新框架，它通过以协作方式利用现有的多模态大型语言模型 (MLLM) 和多个视觉专家模型自动生成高质量的图像描述，从而最大限度地将视觉信息转换为文本

  • 为了解决目前缺乏详细描述基准的问题，提出了几个综合评估基准，以验证我们框架创建的图像描述的质量

  • LLaVA-7B 受益于 IT 策划的描述的训练，获得了更好的生成更丰富图像描述的能力，大大增加了输出的长度和细节，同时减少了幻觉

Introduction

最近的研究强调，图像描述的质量对 MLLM 性能至关重要

  • 低质量的描述经常会在图像理解任务中引起幻觉
  • 具有更丰富视觉概念的详细描述可显著提高生成模型的性能

高质量的图像描述应该传达与相应图像相同的信息，有效地将视觉内容文本化。然而，当前的数据集往往存在不足：

  • 网络抓取的图像文本对，规模大但质量低且噪声大
  • 人工标记的数据集，缺乏深入的细节，制作成本高昂

图像中的信息与其文本描述之间存在很大差距

  • Dalle-3 已进行早期尝试，使用通过 MLLM 生成的描述来训练文本到图像的扩散模型，这表明生成的图像质量有所提高。

  • MLLM 有几个弱点，例如众所周知的视觉幻觉问题和缺乏细粒度细节即使最强大的 MLLM，例如 GPT4-V，也表现出这些弱点。因此，仅依靠 MLLM 来生成数据集仍然有很大的局限性

与此同时，我们注意到计算机视觉领域取得了显著进展，例如物体检测、密集字幕和实例分割。

与使用低分辨率图像（CLIP的默认设置是 336x336）和图像级注释训练的 MLLM 相比，这些视觉专家模型使用高分辨率图像和专门针对感知任务的细粒度对象级注释进行训练，这使得它们能够识别详细内容。然而，这样的模型通常不具备整体理解能力，因此仅依靠这样的模型构建描述是不切实际的。

因此，一个有趣的想法出现了：我们能否将 MLLM 的理解能力与视觉专家的感知能力结合起来，以生成既细节丰富又没有幻觉的高质量描述

Image Textualization(IT)

  1. 整体文本化：我们利用 MLLM 创建参考描述，尽管它缺乏细节并包含幻觉，但它不仅为视觉信息提供了基本结构，也为语言表达提供了基本结构
  2. 视觉细节文本化：然后，我们借助视觉专家模型强大的感知能力来提取细粒度的对象级信息，并将其转换为文本格式。此阶段从图像端提取多个细节并识别参考描述中包含的幻觉
  3. 文本化重新描述。最后，我们利用 LLM 卓越的理解和推理能力，根据前两个阶段的文本信息生成准确而详细的描述，使 LLM 无需“看到”图像即可对其进行描述。这种方法避免了基于 MLLM 的重新描述弱点

IT 框架能够创建细节更丰富且没有幻觉的图像描述

贡献：

  1. 我们提出了图像文本化，这是一个无需人工干预即可自动生成详细图像描述的框架，利用 MLLM 的多模态理解、视觉专家的细粒度感知和 LLM 的推理能力
  2. 我们创建了评估基准并进行了广泛的实验来验证我们框架的有效性。结果表明，生成的图像描述准确捕捉了丰富的视觉细节。
  3. 使用我们的图像文本化框架，我们整理了一个名为 IT-170K 的大规模高质量图像描述数据集。

Method

图像文本化自动生成高质量的图像描述。

  1. 给定一张图像，强大的 MLLM 首先生成一个模板描述，以捕获整体图像内容。
  2. 然后，各种视觉专家模型协作提取模板描述中可能缺少的详细对象信息。
  3. 最后，我们利用强大的 LLM 根据整体信息和细粒度细节重新生成描述。

Holistic Textualization

目前最先进的 MLLM 擅长生成包含更丰富信息和上下文理解的图像描述，相比传统字幕模型生成的描述更是如此。

我们首先利用 MLLM 生成的描述将图像的整体内容文本化，如上图阶段 (A) 所示。尽管存在幻觉和缺乏细节等缺点，但这种描述可以作为具有相对较好结构的基本模板来描述图像。
此后，我们将此描述称为“参考描述”

这种参考描述有两个关键目的：

  1. 从视觉信息方面来看，它包括图像中存在的主要对象和场景的上下文信息。这些元素充当“锚点”，指导后续阶段纳入更多细节。
  2. 从语言表达的角度来看，MLLM 固有的理解和逻辑能力有助于形成组织良好的描述。例如，参考描述通常包括图像的整体描述，然后是主要对象的详细信息，最后以总结性句子结束。与传统的字幕模型相比，这种描述更具逻辑性，表达更自然，这是描述质量的关键因素。

Visual Detail Textualization

第一阶段生成的参考描述通常缺乏视觉细节，并且包含幻​​觉。
在此阶段，我们利用视觉专家模型从图像和参考描述中提取信息。
从图像中，我们捕捉到更多的视觉细节，从参考描述中，我们识别出幻觉内容。最后，我们将细粒度的视觉信息和幻觉对象文本化，如上图阶段 (B) 最右边的灰色框所示。

Hallucination Detection

为了识别参考描述中存在的幻觉，我们首先从中提取对象实体（对象名词和短语）。

在这里，我们利用大型语言模型 (LLM) 强大的指令跟随能力。
具体来说，我们精心设计了一个实体提取提示，并手动注释上下文学习示例，以提高 LLM 的指令跟随能力。

之后，我们使用开放集对象检测器（例如 Grounding Dino）来验证每个提取的实体短语与图像中的对象。
任何在图像中未找到的幻觉对象短语都被标记为“幻觉”，以便在后期删除。 【只考虑了未找到？】

Fine-Grained Objects Annotation

Dense Caption Generation

为了识别原始描述中可能缺失的对象，我们采用了密集字幕器 (DC)

它不仅提供指示对象位置的准确边界框，还将它们与基本属性（例如对象类型、形状和颜色）相关联。

与仅预测对象类别（例如猫）的对象检测器相比，
DC 提供了更详细的描述，例如“一只灰白相间的猫”；

与仅预测图像级字幕的传统字幕模型相比，
DC 能够预测图像中所有可见对象的描述。

这些吸引人的特性使 DC 成为最大化对象信息文本化的合适选择，
这对后续的重新字幕阶段大有裨益。

Spatial Information Collection

现在，我们已经获得了图像中各种对象的密集标题及其边界框坐标。
然而，与原始图像的信息相比，文本化的信息仍然不足。

最关键的原因是，当前的文本化只能传达 2D 平面上对象的相对左右关系，并且可能导致重新描述错误。

例如，考虑一张背景中有一辆汽车而前景中有一个人的图像。他们的边界框可能具有非常接近的坐标。如果我们仅使用这些信息进入重新描述阶段，LLM 可能会使用其逻辑功能错误地将场景描述为“一个人站在汽车旁边”。

发生这种情况的原因是当前的文本化无法捕捉 3D 空间上下文，例如深度（表示对象的前后关系），这对于准确和全面的图像描述至关重要。

  1. 我们首先使用单目深度预测模型获取距离图，其中每个像素的值表示其与相机的距离，从而得出此深度信息。
  2. 我们使用密集字幕器生成的对象边界框，通过 SAM 生成对象分割掩码，从而给出对象的精确像素
  3. 最后，通过平均其相应分割掩码内的深度值来获得对象深度

这样，文本化的信息可以传达对象与相机之间的距离，从而有效地将 2D 密集字幕转换为 3D 字幕

在此过程中，我们注意到两个重要因素：

  1. 利用像素掩码进行尺寸计算：计算对象尺寸的一种简单方法是使用边界框。但是，边界框
    仅提供两个对角，这对于形状不规则的物体（例如长棍）来说可能不准确。如果我们仅依靠边界框的覆盖范围作为对象的大小，则可能导致严重的高估。因此，使用像素级掩码来准确表示对象的大小至关重要。

  2. 规范化：我们对边界框坐标和深度分数进行规范化，以确保值是相对的，这有助于 LLM 在重新捕捉阶段适应不同的图像。

Textualized Recaptioning

我们利用从前几个阶段收集的已转换为文本表示的综合信息，通过 LLM 重建图像描述

包含整体描述和广泛的对象级信息使 LLM 能够准确地解释整个图像及其组成对象，从而无需直接视觉输入

我们在附录中展示了我们方法的有效性，我们精心设计了提示并提供了一些示例来指导 LLM 的生成过程
这确保了 LLM 可以有效地整合新对象，同时最大限度地减少幻觉内容的存在

Experiments

由于缺乏针对长图像描述的标准评估基准，我们首先提出了 DID-Bench、D2I-Bench 和 LIN-Bench 来全面评估详细描述

然后，我们进行了一系列实验来验证我们的图像文本化的有效性，该文本化可以生成高质量的描述

之后，我们验证使用我们框架生成的数据训练 MLLM 可以增强其功能

最后，我们进行语言评估并对我们发布的数据集进行统计分析

Benchmarks and Evaluations

Benchmarks

DID-Bench

详细图像描述基准 (DID-Bench) 包含 200 个样本

  1. 首先，我们利用 MLLM 生成参考描述。为了避免不同 MLLM 的输出习惯带来的偏差，在生成参考描述时，我们对 100 个样本使用 GPT4-V，对剩余的 100 个样本使用 LLaVA

  2. 然后，我们手动检查这些描述的正确性，添加缺失的细节，并删除幻觉内容以建立人工标记的地面真实描述

稍后，我们将使用 LLaVA 参考描述获得地面真实性的分区称为 GT-{LLaVA}，将另一个分区称为 GT-{GPT4-V}

我们采用基于参考的图像描述指标，包括 BLEU、ROUGE-L、METEOR、SPICE和WMD。
这些指标评估生成的描述的各个方面，例如 n-gram 重叠、召回率、准确率、语义内容以及与人工标记描述的整体相似性

D2I-Bench

我们提出描述到图像基准 (D2I-Bench) 来评估描述所捕获的图像信息的完整性。

  1. 首先，我们将描述提供给预先训练的文本到图像模型 (例如 PixArt) 并获得生成的图像
  2. 然后，我们使用来自预先训练的 CLIP的图像编码器提取原始图像和生成图像的图像嵌入
  3. 我们计算图像嵌入之间的余弦相似度

较高的相似度得分表明描述有效地捕捉了图像细节，从而生成与原始图像非常相似的图像

LIN-Bench

为了全面评估描述的可读性和语言特征，我们提出了
语言基准 (LIN-Bench)，它采用了 ARI、FK 和 SMOG 等指标。

如果句子中的单词更多且字符更多，ARI 往往会更高，而 FK 和 SMOG 则更注重多音节单词的数量。更详细的描述往往会在这些指标中获得更高的值。

POPE

评估 MLLM 所遭受的幻觉程度，其中包括有关图像中物体存在的问题，并附有“是”或“否”等简短答案

Image Description Quality Evaluation

DID-Bench Results

为了验证我们的图像文本化注释框架在生成详细而准确的图像描述方面的有效性，我们将图像文本化生成的描述的质量与 MLLM 直接生成的描述的质量进行了比较

我们观察到不同 MLLM 和地面实况注释的所有指标都有显著提高

有趣的是，我们观察到 IT 生成的描述的评估结果也取决于整体文本化阶段使用的 MLLM。这是因为评估指标不仅考虑了视觉正确性，还考虑了描述的风格，例如代词和介词

因此，为了排除语言偏见的影响，使用不同风格的 GT 注释进行评估至关重要。

D2I-Bench Results

我们的 IT 框架生成的描述产生的图像与原始图像的相似度得分高于 COCO 的描述和 MLLM 生成的描述

我们提供了定性示例，以显示 IT 生成的描述产生的图像与原始图像更相似。这些结果证明了我们的框架在准确捕捉图像内容细节方面的有效性。

MLLM Tuning Evaluation

DID-Bench Results

我们比较了 1) 原始 LLaVA-7B 模型生成的图像描述的质量；2) 使用 MLLM 生成的描述进行微调的 LLaVA 和 3) 使用图像文本化生成的描述进行微调的 LLaVA。

我们观察到，使用 IT 策划的数据集进行微调的 MLLM 在所有指标和 GT 注释中始终优于其他基线

  1. GT-LLaVA 上的得分大多高于 GT-GPT4V，这是 LLaVA 的响应方式
  2. 对于每个 GT 分割，IT 调整后的 LLaVA 的表现都大大优于基线和 MLLM 调整后的 LLaVA
  3. 从组合 GT 的评估来看，我们观察到 IT-LLaVA 的有效性接近 GPT4-V，而 IT-GPT4-V 仍然明显超过所有同行。这表明图像文本化有潜力缩小不同 MLLM 之间的能力差距

POPE and LIN-Bench Results

左侧，我们评估了使用不同图像描述数据调整的 MLLM 的幻觉。我们观察到，使用 IT 生成的描述进行调整可以最显著地缓解幻觉。

在右侧，我们还展示了 LIN-Bench 上的结果，这表明使用 IT 生成的描述进行调整可以在生成包含更丰富细节的描述方面获得最大的收益。

Linguistic-based Evaluation

Statistical Analysis

我们分别总结了图像文本化和 MLLM 基线生成的图像描述的统计数据。

显示 IT 生成的描述的字数和句子数高于 MLLM 生成的描述。

我们显示了不同类型单词的计数，这表明 IT 生成的描述包含更多单词，例如名词、动词和形容词。这些统计分析表明，IT 生成的描述从图像中捕获了更全面的视觉信息。

LIN-Bench Results

IT 生成的描述在所有指标上都获得了更高的分数，这表明这些描述能够从图像中包含更丰富的视觉细节

Conclusion

总之，本文解决了现有图像描述数据集的局限性，并提出了一个创新框架，即图像文本化 (IT)，以生成详细而准确的图像描述。

该框架以协作的方式利用多模态大型语言模型 (MLLM) 和多个视觉专家模型的强大功能。
通过对图像理解和生成任务的大量实验，我们验证了该框架生成的描述的高质量。
我们希望我们的工作能够为设计更高效、更可扩展的方法来生成详细而准确的图像描述提供灵感。
